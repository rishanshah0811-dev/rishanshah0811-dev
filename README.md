## Hey, I'm Rishan

CS student at Northeastern University. I build full stack apps — mostly with TypeScript, Next.js, and Python. Right now I'm focused on AI-powered tools that solve real problems.

### What I've been building

**[Recall](https://github.com/rishanshah0811-dev/recall-ai)** — An AI assistant with persistent memory. It extracts facts from conversations, stores them as vector embeddings in Qdrant, and uses them to personalize future responses. The more you talk to it, the better it knows you. [Live demo](https://recall-ai-rishanshah0811-devs-projects.vercel.app)

**[AI Interviewer](https://github.com/rishanshah0811-dev/ai-interviewer)** — Real-time mock interview simulator. Pick a role, speak your answers out loud, and get scored instantly using Gemini and the Web Speech API. [Live demo](https://ai-interviewer-nu-tan.vercel.app)

**[NL-SQL Analyzer](https://github.com/rishanshah0811-dev/nl-sql-analyzer)** — Ask questions about data in plain English. It generates SQL, runs it, and returns results with charts. Built with FastAPI and Next.js. [Live demo](https://nl-sql-analyzer.vercel.app)

**[TrendSense](https://github.com/rishanshah0811-dev/trendsense)** — A trend analysis dashboard. [Live demo](https://trendsense-iota.vercel.app)
